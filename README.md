# AIUB Grading System (CGPA Calculator)

A simple, dark-mode CGPA calculator web UI made as an AIUB student project.

## Disclaimer
This is not AIUB's official website. It is a project developed by an AIUB student.

## Project Structure
- `frame/` — Main web UI (`index.html`, `styles.css`, `script.js`)
- `assets/` — Static assets (e.g., `aiub-logo.png`)
- `source/` — Java source (optional/legacy)

## Run Locally
You can open the HTML directly, but using a local server is recommended.

## Notes
- Enter your current CGPA and completed credits, then add your current-semester courses to calculate Semester GPA and updated CGPA.
